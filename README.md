# Webserver — Portfolio

Een statische portfoliowebsite gebouwd met HTML, CSS en JavaScript. De site bevat ook ondersteunende documenten en PDF-bestanden.

## Projectstructuur

```
webserver5/
├── index.html          # Hoofdpagina
├── css/
│   └── style.css       # Stijlen
├── js/
│   └── main.js         # JavaScript logica
├── docs/               # Word-documenten
└── pdf/                # PDF-bestanden
```

## Lokaal openen

Geen installatie nodig. Open gewoon `index.html` in je browser:

```bash
# Via bestandsverkenner
Dubbelklik op index.html

# Of via terminal
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```
